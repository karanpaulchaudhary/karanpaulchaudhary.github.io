Questions

  • Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

    • The best way to explain HTML is to think about car ,the car has defined shape,it has windows ,doors and position for everything.Now css is the internal decoration of car ,leather seats ,space between seats,color of carpet etc.While DOM is the visual representation of car which we can change virtually to see how it will look like .without changing real one.

  • What is meant by boxifying design?

    • Boxifying design means represting design by dividing into boxes.The outer box can be the title and next box inside it has top as header and bottom as footer and internal box as heading,text etc.All this by side by side defining the magin of every box and space it will take.

  • What is the box model?

    • The Box Model is basically a box that wraps around every single HTML element, and is made up of:

    content

    padding

    border

    margin

    All these can be given values to dictate how the box will look.
